Il existe plusieurs types de quêtes secondaires :

  • exploration
  • escorte
  • chasse aux monstres
  • prime
  • assassinat (uniquement pour un joueur ayant la classe Assassin)
  • videur (uniquement dans les tavernes, auberges et bordels)
Les quêtes ont 4 niveaux de difficultés :
  • facil
  • normal
  • difficil
  • rude
  • extrême

Les quêtes secondaires peuvent se récupérer dans plusieurs lieux :

  • la taverne
  • l'auberge
  • le bordel
  • la garnison
Ces différents lieux sont présents dans tout les villages mise à part dans les tout petits villages (une centaine d'habitants) où la taverne, l'auberge et le bordel sont rassemblés dans le même bâtiment.

La taverne :

La taverne est le lieu les habitants et voyageurs viennent pour boire et jouer à des jeux de hasards. Les joueurs peuvent y récupérer des quêtes :
  • d'exploration
  • d'assassinat
  • de videur

L'auberge :

L'auberge est le lieux où les voyageurs viennent pour se nourrir et prendre du repos. Ce lieux et par conséquent beaucoup plus calme que la taverne et le bordel. L'auberge compte parmis sa clientelle principalement des marchands. Les joueurs pourront récupérer des quêtes :
  • d'exploration
  • d'escorte
  • de videur

Le bordel :

Le bordel est l'endroit ou les habitants et les voyageurs viennent chercher de la compagnie auprès d'un personnel au charme négociable. Les joueurs peuvent y trouver les type de quêtes suivantes :
  • tous les types de quêtes
  • de videur

Quêtes de primes
Ce type de quête consiste à traquer un bateau ou un groupe de bandit.
Quêtes d'assassinat
Ce type de quête n'est accessible pour que pour un personnage ayant la classe assassin. Les récompenses sont attribués uniquement au joueur réalisant la quête.
Quêtes de videur
Ce type de quête n'est disponible que dans les établissements de types auberge, taverne et bordel. Dans ces lieux où l'on boit de l'alcool, prend des paris, etc les humeurs peuvent s'échauffer. Les gardes étant rarement présent pour maintenir l'ordre dans ces lieux, les gérants de ce type d'établissement peuvent demander l'aide des joueurs pour mettre dehors et corriger les fauteurs de troubles.
La difficulté de ce type de quête est fixée à normale. Récompenses :
Expérience PA Renom Moral
150 Consommation gratuite pour le reste du séjour, y compris le prix des chambres 5 1
